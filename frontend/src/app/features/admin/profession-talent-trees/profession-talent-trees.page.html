Process AuctionHelper SavedVariables locally

Select your SavedVariables folder. The browser finds AuctionHelper.lua and AuctionHelper_Professions.lua, then processes only those matching files on this device.

Raw Lua never leaves this device. Other files in the folder are ignored without being read, and only normalized JSON is sent after you review and submit it.

Replacing talent-tree definitions is destructive

Local processing and server validation only report available data. They do not import or replace shared talent-tree definitions.

SavedVariables does not contain a reliable region. Enter it so identical character names on different regions stay separate.

Choose the SavedVariables directory itself. Only the two required filenames are selected.

Can’t select a folder?

Choose the two files individually instead.
